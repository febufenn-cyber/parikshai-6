# Phase 0 research workspace

Research is evidence, not marketing material.

## Structure

- `interviews/` — guides and de-identified notes.
- `prototype-feedback/` — task results, comprehension, defects, and return behaviour.
- Future folders may include `competitive-notes/`, `channel-tests/`, and `economics/`.

## Research rules

- Record what the learner currently does before asking what they might use.
- Separate verbatim observation from interpretation.
- Do not store unnecessary personal data.
- Mark sample bias and recruitment source.
- Track contradictions, not only confirming quotes.
- Count commitments and repeated use more strongly than compliments.
- Never fabricate interviews to satisfy an entry gate.

Each session note should include date, participant segment, current workflow, pain evidence, trust/language findings, commitment signal, surprises, and implications.
