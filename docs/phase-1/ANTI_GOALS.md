# Phase 1 anti-goals

Phase 1 deliberately does not:

- claim that the TNPSC Group IV wedge is market-validated;
- import unreviewed production questions;
- expose canonical answers to learner clients;
- compute learner mastery;
- implement live AI generation;
- build a full editorial UI;
- automate current affairs;
- make voice or TTS a dependency;
- promise database tests passed before they run in a Supabase/Postgres environment;
- allow code completion to override the Phase 0 evidence gates.

These constraints preserve the move order: trustworthy content infrastructure first, learner loop second, mastery third, grounded tutoring fourth, and controlled generation only after validation exists.
